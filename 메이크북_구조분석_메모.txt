■ 프로젝트 개요
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
철학 아트북 제작기
- 탈무드·니체·쇼펜하우어 3대 철학 주제
- 주제당 1000개 명언 → AI 본문 생성 → 이미지 첨부 → 인쇄용 아트북
- 기술: React 18 + Babel (CDN), Node.js 서버, 로컬 Ollama
- 실행: node server.js → http://localhost:8787/
- 클라우드/외부망 완전 차단 (100% 로컬)


■ 시리즈 규모
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 주제당 200권 × 5편 = 1000개 명언 전수 사용 (중복 0)
- 총 3주제 × 200권 = 600권 목표
- 1권 = 5편 본문 (각 본문 = 명언 + AI글 + 4컷만화 + 상징 일러스트)

  data.jsx (6KB)       — TOPICS(3주제), BOOK_SPREADS(6스프레드), 카테고리 한→영

[AI / 프롬프트]
  ai-writer.jsx (21KB) — Ollama 본문 생성 엔진
                         · 한자/가나 검출 → 최대 4회 재생성
                         · 12줄 미만이면 확장 패스
                         · 데모 폴백 (Ollama 미가동 시)
  prompt-generator.jsx (23KB) — 이미지/4컷 프롬프트 빌더 페이지
                         · 옵션 선택 → 영어 프롬프트 자동 조립 → 클립보드 복사
  prompt-data.jsx (19KB) — 캐릭터 7종, 배경 30종, 시간·분위기·날씨·조명·카메라·화풍
  prompt-options-data.jsx (43KB) — 집 스타일별 상세 프롬프트 데이터

[뷰 / 렌더링]
  book-view.jsx (115KB) — 책 구조 탭 + 미리보기 + PDF 출력
                         · A4 인쇄 임포지션 (12장 레이아웃)
                         · html2canvas + jsPDF로 PDF 생성
                         · Ollama 한→영 번역 (영문 PDF용)
                         · 한/영 PDF + 한/영 이미지 4개 일괄 생성
  spread-builder.jsx (12KB) — 스프레드 미리보기, 이미지 첨부 슬롯

[데이터 관리]
  persist-store.jsx (2KB) — IndexedDB KV 스토어 (artbook_db)
  usage-tracker.jsx (18KB) — 명언 사용 대장 (중복 0 추적) + QuotePicker
  recovery.jsx (18KB)      — 자동저장 복구 패널
  tweaks-panel.jsx (26KB)  — 설정 패널 (4컷 위치, 주제 팔레트, 제본 디테일)

[명언 풀]

  quotes-nietzsche.jsx (~67KB)    — 니체 10카테고리 × 100개
  quotes-schopenhauer.jsx (~67KB) — 쇼펜하우어 10카테고리 × 100개

[스타일]
  styles.css (64KB)        — 메인 스타일시트

  quotes-styles.css (7KB)  — 명언 피커 스타일


■ 서버 API (server.js)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  POST /ollama/chat   → Ollama 프록시 (브라우저 CORS 우회)
  POST /save-page     → 페이지 파일 저장 (txt + 이미지 b64)
  POST /save-pdf      → PDF 파일 저장 (base64 → 파일)
  POST /delete-page   → 페이지 삭제
  GET  /*             → 정적 파일 서빙

■ 인쇄 임포지션 (A4 양면, 12장)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  ① 철학종 (커버, 250g 별도)
  ②~⑥ 앞면 5장 (본문 001_a~010_b)
  ⑦~⑪ 뒷면 5장 (목차→줄공책→줄공책→줄공책→명언)
  ⑫ 줄공책 (별도 시트, 사장님이 여러 장 인쇄)

  인쇄 순서: 5장 앞면 인쇄 → 뒤집어서 5장 뒷면 인쇄 → 실 노출 제본


■ UI 탭 구조
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  [작업실]     — 3패널: 왼쪽(주제/카테고리/명언/AI작성) | 가운데(미리보기) | 오른쪽(첨부/MAKE)
  [책 구조]    — 전체 스프레드 그리드 + 인쇄 구조 맵 + PDF 출력
  [미리보기]   — 실제 책처럼 넘기며 보기 + 본문 인라인 편집
  [사용 현황]  — 주제별 명언 사용 대장 (권별 카드)
  [복구]       — IndexedDB/디스크 백업에서 작업물 복원


■ 작업 워크플로우
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. 주제·카테고리 선택
  2. 명언 풀에서 구절 선택 (미사용 필터 가능)
  3. AI 본문 생성 (Ollama qwen2.5:14b) → 버전 관리
  4. 이미지 프롬프트 페이지에서 프롬프트 생성 → 클립보드 복사
  5. 외부 이미지 생성 도구에서 이미지 제작
  6. 4컷 카툰(9:16) + 상징 일러스트(1:1) 첨부
  7. MAKE 버튼 → 스프레드 완성 + 명언 대장 등록
  8. 5편 완성 후 → PDF 출력 (한/영 동시)
  9. 인쇄 → 실 노출 제본

■ 자동저장 시스템 (3중 백업)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. IndexedDB (브라우저) — 상태 변경 후 700ms, 전체 스냅샷 (글+이미지)
  2. 디스크 자동저장 — 5분마다, pages/<주제>/<권>/autosave/
  3. 페이지별 백업 — MAKE 시, 페이지당 최근 20버전

  워크스페이스 키: workspace_{주제명}_{NNN}권 (예: workspace_탈무드_001권)
  마이그레이션: 구형식(workspace_N권) → 신형식 자동 변환

  - 같은 (book, spread) 자리는 덮어쓰기 허용
